Créer un compte, se connecter et récupérer sa clé API

Ces premières étapes se font depuis l'interface web de BabelNotif, pas par appel API — c'est un humain qui les fait une seule fois, avant que votre backend n'appelle quoi que ce soit.

  1. Créer un compte

    Depuis la page d'accueil, cliquez sur "Commencer". Le formulaire d'inscription demande :

    • Nom de l'application — ex : websms
    • Email de contact
    • Téléphone
    • Mot de passe et sa confirmation
    • L'acceptation des conditions d'utilisation

    Le compte est créé avec un quota de 0, en attente de vérification de votre identité (étape suivante).

  2. Vérifier votre identité (OTP)

    Après l'inscription, choisissez le canal de vérification (email ou SMS) puis entrez le code reçu. Cette étape confirme que l'email/téléphone fourni vous appartient bien.

    Essai gratuit automatique. Dès la vérification réussie, un essai de 30 jours est activé sans paiement — 100 notifications incluses, tous canaux confondus. Aucune carte ni Wave requis pour commencer à tester l'API.

  3. Se connecter

    Depuis la page "Se connecter", entrez votre email de contact et votre mot de passe. Vous accédez alors à votre dashboard.

  4. Retrouver votre clé API

    Dans le dashboard, section Paramètres → Informations du compte, votre clé API est affichée dans un champ dédié (format bn_live_...), avec un bouton pour la copier directement.

    Gardez cette clé secrète. Ne la partagez jamais publiquement — c'est elle qui identifie et authentifie votre backend auprès de BabelNotif (voir Intégrer l'API).

  5. Souscrire à un forfait (après l'essai gratuit)

    Une fois les 30 jours ou les 100 notifications de l'essai épuisés, passez à un forfait payant depuis l'onglet Forfaits du dashboard, réglé via Wave (mobile money) :

    ForfaitPrixQuota SMS + Email
    Starter100 FCFA / mois1 000 notifications
    Pro15 000 FCFA / mois5 000 notifications
    Business35 000 FCFA / mois20 000 notifications
    Push et WebSocket sont illimités et inclus dans tous les forfaits — seuls les canaux SMS et Email consomment votre quota mensuel.

    Le paiement est confirmé automatiquement dès validation côté Wave, et votre quota est activé sans action supplémentaire de votre part.

Et après ?

Avant de tester ou d'envoyer la moindre notification, créez au moins une règle (depuis le dashboard ou par API) — sans règle correspondant à votre événement, /notify refusera l'appel. Une fois la règle en place, direction Intégrer l'API pour votre premier appel, cette fois-ci en code.
